# Discord Wrapped 2025 - Frontend

React + TypeScript + Tailwind CSS + Framer Motion frontend for Discord Wrapped 2025.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
npm run preview
```

## Features

### Story Mode
- Guided narrative experience
- Act 1: Culture (who we are)
- Act 2: Individual spotlights (all 11 members)
- Awards ceremony
- Bechdel test reveal
- Synesthesia segment
- Partner messages
- Smooth animations with Framer Motion

### Dashboard Mode
- Explorable stats
- Member cards
- Quick stats overview
- (More features coming)

### Data
All data loaded from `src/data/wrapped_data.json` (208 KB)
- 139,215 messages analyzed
- 11 complete profiles
- 12 awards
- Bechdel test
- Synesthesia colors & evidence
- Inside joke timelines
- And more!

## Tech Stack
- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
